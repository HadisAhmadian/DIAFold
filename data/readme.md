## What’s inside
| Sub-folder | Contents |
|------------|----------|
| `fasta/` | FASTA sequences for 47 CASP14/15 proteins |
| `native_structures/` | Experimental PDB structures of the same targets |
| `MSAs/` | Multiple-sequence alignments (built once with **DIAMOND**, once with **MMseqs2**) |
| `predicted_structures/` | AlphaFold models generated via **ColabFold** |
