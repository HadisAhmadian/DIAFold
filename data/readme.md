# data/

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.15420896.svg)](https://doi.org/10.5281/zenodo.15420896)

## What’s inside
| Sub-folder | Contents |
|------------|----------|
| `fasta/` | FASTA sequences for 47 CASP14/15 targets |
| `native_structures/` | Experimental PDB structures (same targets) |
| `MSAs/` | Multiple-sequence alignments (generated once with **DIAMOND** and once with **MMseqs2**) |
| `predicted_structures/` | AlphaFold models made via **ColabFold** |
